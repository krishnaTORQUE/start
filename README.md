# Cdefer start

## Cdefer install / update

```bash
curl -L https://raw.githubusercontent.com/krishnaTORQUE/cdefer/refs/heads/main/install.sh | sh
```

## Start / run project

```shell
git clone https://github.com/krishnaTORQUE/start
cd start
cdefer run
```
